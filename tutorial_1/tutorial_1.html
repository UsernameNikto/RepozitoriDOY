Браузеры

Прикладное программное обеспечение для просмотра страниц, содержания веб-документов, компьютерных файлов и их каталогов; управления веб-приложениями; а также для решения других задач.

Ресурсы

Библиотека более 800 свободно распространяемых шрифтов, интерактивный каталог для их просмотра, и прикладные программные интерфейсы для использования веб-шрифтов посредством CSS и на Андроиде.

Технологии

Методология web-разработки, а также набор интерфейсных библиотек, фреймворков и вспомогательных инструментов.

Редакторы кода

Текстовый редактор для создания и редактирования исходного кода программ. Он может быть отдельным приложением или встроен в интегрированную среду разработки
